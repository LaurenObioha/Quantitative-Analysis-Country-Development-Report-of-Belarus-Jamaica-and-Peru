# Quantitative-Analysis-Country-Development-Report-of-Belarus-Jamaica-and-Peru
# 📋 Overview 
This project reports the country development of Belarus, Jamaica and Peru from 1980-2020.

# 📌 Objective
To assess the determinants of economic development in Belarus, Jamaica and Peru using Statistical analysis. 

# 📈 Analysis Workflow 
- Imported and inspected datasets from the World Bank.
- Computed descriptive statisctics and visualized data.
- Conducted regression analysis and computed summary statisctics
- Interpreted statistical results.
- Reported findings and conclusions.

# 🔍 Example Result 1
- Null hypothesis (H0): Trade has no explanatory power for model M4.
- Alternative hypothesis (H1): Trade has an explanatory power for model M4.

After performing  t-test:
- p-value > 0.05 -> Accept H0
- Conclusion: It can be suggested that trade has no explanatory power on M4 and Trade has a non-significant effect on GDP.

# 🔍 Example Result 2
- Null hypothesis (H0): Model 8 (M8) does not affect Gross Development Product (GDP).
- Alternative hypothesis (H1): Model 8 (M8) has an effect on Gross Development Product (GDP).

After performing t-test:
- p-value < 0.05 -> Reject H0
- Conclusion: M8 significantly affects GDP suggesting that Model M8 has the largest explanatory power for economic development, holding all other variables constant. 

# 💡 Key Learnings 
- Applying hypothesis testing to real-world data.
- Understanding how sample means compare to population benchmarks.
- Communicating findings through staistical reasoning and visuals.
- Reproducible workflows for academic-style data analysis.
